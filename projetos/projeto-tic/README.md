# Estrutura de Tecnologia da Informação (TIC)

## Descrição

Projeto acadêmico desenvolvido com foco na análise e organização de sistemas de Tecnologia da Informação.

## Objetivo

Realizar o mapeamento e estruturação de informações relacionadas aos sistemas de uma organização.

## Atividades desenvolvidas

- organização de informações;
- análise de sistemas;
- estruturação de TI;
- documentação do projeto.
